# StudyFlow 📚✨

StudyFlow is a simple landing page project built to practice **Tailwind CSS** while designing a modern, AI-inspired study planner interface.  
The goal of this project is to improve frontend layout skills, responsiveness, and UI structure using utility-first CSS.

This is currently a **landing page only**, focused on visual design and layout.

---

## 🛠 Technologies Used

- **HTML5**
- **Tailwind CSS**
- **JavaScript** (basic interactions, e.g. mobile menu)
- **Vite** (for local development)

---

## ✨ Features (Planned & Current)

### Current
- Responsive landing page layout
- Hero section with call-to-action
- Navigation bar with mobile menu
- Feature, About, and How It Works sections
- Modern glassmorphism-inspired UI

### Planned
- More refined animations and transitions
- Improved accessibility
- Additional sections (future dashboard preview)
- Possible backend integration (later stage)

---

## 🧠 Our Process

This project started as a way to **practice Tailwind CSS**.

The idea came from asking for **project suggestions to build with Tailwind**, and choosing to create a realistic SaaS-style landing page instead of small UI components.

Process so far:
1. Chose a project idea (AI Study Planner)
2. Designed a simple landing page structure
3. Built sections step by step using Tailwind utilities
4. Focused on layout, responsiveness, and clean UI
5. Iteratively improved based on visual feedback

The focus is on **learning by building**, not rushing features.
